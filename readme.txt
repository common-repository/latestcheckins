Always your latest Foursquare checkins in your sidebar!

== Description ==

With this plugin, you can add an widget to your sidebar which is showing you your latest Foursquare check-ins.
Plugin is Dutch!

== Installation ==

1. Extract the ZIP and upload the content to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Setup the plugin
4. Add the widget add your sidebar

== Frequently Asked Questions ==

= Do i need to enter my user credentials =

Nope, we only need your RSS url

= Where can i find my RSS url =

https://www.foursquare.com/feeds/

= How does the RSS url looks like =

Something like this: https://feeds.foursquare.com/history/333333XXXXAAAAAMMMMMPPPP1111133333.rss
